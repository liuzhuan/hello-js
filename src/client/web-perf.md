# 如何使用网络性能 API？

[A Primer for Web Performance Timing APIs](http://w3c.github.io/perf-timing-primer/), by *Xiaoqian Wu*, 2018/10/17

`window.performance` 全局对象包含资源时间信息。

使用 `performance.getEntriesByType('resource')` 可以获取每个请求资源的详细时间信息。
